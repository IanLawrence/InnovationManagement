setRowAttribute(rId, name, value)

Required library edition:
Required library file: dhtmlxgrid.js

sets row attribute

  • rId - row ID
  • name - attribute name
  • value - new attribute's value
   <?xml version='1.0' encoding='iso-8859-1'?>
        <rows>
            <row id="unique_rowid" some="data">
                    <cell some="data">cell content</cell>
            </row>
        </rows>
   ...
   var rowAttr = grid.setRowAttribute("unique_rowid","some","new value");

See also: getRowAttribute
