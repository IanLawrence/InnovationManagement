attachFooter(values, style)

attachFooter(values, style)

Required library edition:
Required library file: dhtmlxgrid.js

attach additional line to footer

  • values - array of header titles
  • style - array of styles, optional
        //plain usage
        grid.attachFooter("A,B,C,D");
        //alternative syntax
        grid.attachFooter(["A","B","C","D"])
        //colspans in footer
        grid.attachFooter("A,#cspan,C,#cspan");
        //rowspans in footer
        grid.attachFooter("A,#rspan,C,#rspan");
        //styles assigned
        grid.attachFooter("A,B,C,D",["","color:red;","",""]);

attachFooter() command supports HTML as values, so it may contain any HTML compatible content. 

      grid.attachFooter("A,<strong>B</strong>,C,<a href='http://dhtmlx.com'>D</a>");

If you need to call attachFooter() after grid was loaded you should call setSizes() command: 

   grid.load("grid.xml",function(){
        grid.attachFooter('A,B,C');
	grid.attachFooter('D,E,F');
	grid.attachFooter('G,H,I');
        grid.setSizes();
   });
